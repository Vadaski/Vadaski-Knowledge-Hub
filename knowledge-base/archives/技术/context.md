# 技术分类目录

## 目录概述
此目录用于归档技术相关的文章和资源，包括AI、软件工程、编程语言、架构设计等技术主题。

## 子目录
- **AI**: 人工智能相关技术文章和资源，包括机器学习、深度学习、AI系统设计等。[链接](./AI/context.md)
- **软件工程**: 软件工程方法论、开发实践、项目管理等相关内容
- **编程语言**: 各种编程语言的技术文章和教程
- **架构设计**: 系统架构、设计模式、技术架构等相关内容

## 规则
- 技术分类下的内容应按照具体技术领域进一步细分
- 每个子分类应维护独立的 context.md 文件
- 文件命名格式：`YYYY-MM-DD-文章标题.md`
