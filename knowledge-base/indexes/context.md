# 索引目录上下文

## 目录概述
此目录用于维护知识库的语义索引和检索信息。索引文件帮助快速定位和检索知识库中的内容，支持 context-provider 代理的语义搜索功能。

## 文件摘要
- 当前目录为空，索引将在内容填充后自动生成和维护
- 索引文件可能包含：关键词索引、语义向量索引、内容分类索引等

## 子目录
- (无子目录)

## 模块架构
索引目录采用扁平结构，所有索引文件直接存储在此目录下。索引由 context-provider 代理自动维护和更新。

## 规则
- 索引文件由 context-provider 代理自动生成和维护，不建议手动修改
- 索引更新应遵循增量更新原则，基于 Git 提交差异进行更新
- 索引应保持与知识库内容的同步
- 索引文件应记录生成时间和相关元信息

## AI 协作工作流
1. 初始索引：context-provider 使用 create-context 命令创建初始索引
2. 增量更新：context-provider 使用 update-context 命令基于 Git 差异更新索引
3. 索引检索：knowledge-butler 使用索引快速检索相关知识
4. 索引验证：定期使用 validate-context 命令验证索引完整性

## 索引元信息
- Git 快照: 无
- 最后更新: 2024年（当前索引更新）
- 索引状态: 基础结构已建立，等待内容填充后生成实际索引
